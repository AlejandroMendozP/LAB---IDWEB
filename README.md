1. Clonar el repositorio localmente en la computadora.
2. Asegurarse de que todas las rutas de imagenes, javascript, python, static, etc estan correctamente conectadas.
3. Ejecutar el archivo python que incluye el servidor con wsgi.
4. Entrar al link localhost/8000 en donde se simula y cargan los archivos estáticos.
5. Se puede navegar entre páginas, en primer lugar se ingresa al index.html, donde se pueden observar los resultados del FC Barcelona en la liga de campeones.
6. En segundo lugar, se entra a la página "Historia", en donde hay un menú interno que navega internamente en la página web entre secciones.
7. En tercer lugar, la plantilla, cada jugador tiene la opción de mostrar una imagen más grande en toda la pantalla junto con un botón "Volver" que permite mostrar toda la interfaz de jugadores otra vez.
8. En cuarto lugar, el palmaarés, que muestra una tabla con todos los trofeos del FC Barcelona, y de fondo una imagen con un position.
9. En último lugar, un formulario de socios, en donde se ingresan los datos y se mandan a una base de datos externa, se pueden seleccionar paquetes y cada dato se valida para que esten correctamente escritos o seleccionados.
